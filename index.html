Va para as questões da prova

1. O que são VLANs? Para que elas servem?

VLANs (Virtual Local Area Networks) são redes locais virtuais que segmentam uma rede física em várias redes lógicas distintas. Elas servem para separar domínios de broadcast, melhorar a segurança, a performance da rede, e permitir que dispositivos em diferentes locais físicos, mas na mesma VLAN, se comportem como se estivessem na mesma rede física.

2. Cite três vantagens ao se reduzir o tamanho dos domínios de broadcasts.

Reduzir o tamanho dos domínios de broadcast oferece várias vantagens:

  • Menor Tráfego de Rede: Reduz a quantidade de mensagens de broadcast que circulam na rede, diminuindo a latência e aumentando a eficiência.
  • Melhor Desempenho: Com menos dispositivos por domínio, a rede responde mais rapidamente, pois há menos colisões e congestionamentos.
  • Aumento da Segurança: Limita o alcance dos broadcasts, dificultando que dispositivos mal-intencionados acessem toda a rede.

3. O que são enlaces de trunk?

Enlaces de trunk são conexões entre switches que transportam o tráfego de múltiplas VLANs ao mesmo tempo. Eles utilizam um protocolo de tagging, como o IEEE 802.1Q, para identificar a que VLAN cada quadro de dados pertence, permitindo que múltiplas VLANs compartilhem um único enlace físico.

4. Quantas VLANs é possível ter em um switch? E qual delas é a VLAN nativa?

É possível ter até 4096 VLANs em um switch, mas o limite prático é geralmente menor devido a restrições de hardware e configuração. A VLAN nativa é a VLAN padrão para tráfego não marcado, que normalmente é a VLAN 1, mas pode ser configurada para outra VLAN para melhorar a segurança.

5. Qual equipamento permite segmentar os domínios de colisão? Em qual camada do modelo OSI esse equipamento opera?

Os switches são os equipamentos que segmentam domínios de colisão. Eles operam na Camada 2 (Camada de Enlace) do modelo OSI, utilizando endereços MAC para direcionar o tráfego de dados dentro da rede.

6. Qual equipamento permite segmentar os domínios de broadcast? Em qual camada do modelo OSI esse equipamento opera?

Os roteadores são os equipamentos que segmentam domínios de broadcast. Eles operam na Camada 3 (Camada de Rede) do modelo OSI, utilizando endereços IP para encaminhar pacotes entre diferentes redes.

7. Qual o equipamento necessário para realizar uma comunicação entre VLANs diferentes?

Para permitir a comunicação entre VLANs diferentes, é necessário utilizar um roteador ou um switch de Camada 3 (com capacidade de roteamento). Esse processo é conhecido como Inter-VLAN Routing, e ele permite que pacotes sejam encaminhados entre VLANs distintas.
